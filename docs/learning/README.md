# Learning Logs

このディレクトリは、Morning News の学習ログを置く場所です。

単なる「勉強したこと」ではなく、実装中に出た問題をどう調査し、どう設計へ反映し、何を学んだかを残します。採用側が見たときに、実務に近い改善プロセスを追えることを目的にしています。

HTML版の一覧は `public/pages/learning.html` で確認できます。

## 読んでほしい観点

| 観点 | 見せたいこと |
| --- | --- |
| 問題発見 | 実装中に起きた制約、失敗、見せ方の弱さを言語化したこと |
| 調査 | API制限、ログ、安全性、テスト観点を確認したこと |
| 実装 | 設定、Provider、例外、ログ、テスト、ドキュメントへ落としたこと |
| 学び | 次の設計に使える判断基準として残したこと |
| AI活用 | AIを設計整理、レビュー、テスト観点出しに使い、最終判断は自分で行ったこと |
